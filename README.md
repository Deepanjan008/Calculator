### 🚀 Key Modules

Neo-Calculator comes packed with specialized tools for every need:

* **Core Calculator:** Supports advanced equations, haptic feedback, and custom themes.
* **Unit Converter:** 15+ categories including Length, Area, Volume, and Data.
* **Health Suite:** Integrated BMI and Age calculators.
* **Finance Tools:** Real-time Currency converter and Investment trackers.

### 🛠️ Tech Stack

* **Language:** Kotlin
* **UI Framework:** Jetpack Compose
* **Architecture:** MVVM (Model-View-ViewModel)
* **Design:** Material 3 with Custom Glassmorphism
